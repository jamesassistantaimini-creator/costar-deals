# CoStar Deal Underwriting Map

Interactive map of San Diego retail properties with downloadable rent rolls and offering memorandums.

## Features

- 📍 **Interactive Leaflet Map** — zoom, pan, and cluster properties
- 🔍 **Property Search** — click pins to view details
- 📄 **Rent Rolls & OMs** — linked to each property
- 🎯 **Mobile Friendly** — accessible from any device
- 🔄 **Auto-Updated** — regenerates after each CoStar download batch

## Usage

1. Open the map: https://jamesassistantaimini-creator.github.io/costar-deals/
2. Click any property pin to view details
3. See associated PDF files in the sidebar
4. Click a file to open it

## Data

- **90+ properties** across San Diego retail market
- **325+ rent roll and offering memorandum files**
- **Real coordinates** (geocoded from CoStar property addresses)
- **Auto-synced** with CoStar download pipeline

## Updates

Map regenerates automatically when new properties are downloaded via the CoStar bulk download script.
